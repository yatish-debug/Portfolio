## 🚀 Quick Start

```bash
# Clone repository
git clone https://github.com/Yatish/cyber-portfolio.git
cd cyber-portfolio

# Install backend dependencies
npm install

# Start the Node.js server
node server.js

# Then visit: http://localhost:3000
```

> **Requirements:** Node.js (v14+ recommended), Modern web browser

## 📁 Project Structure

```
cyber-portfolio/
│
├── server.js             # Backend Express server
├── package.json          # Node.js dependencies
│
├── data/                 # 🔒 Secure JSON Databases
│   ├── analytics.json    # Visitor tracking
│   ├── experience.json   # Work history
│   ├── projects.json     # Portfolio projects
│   └── messages.json     # Contact form submissions
│
└── public/               # 🌐 Publicly Accessible Frontend
    ├── index.html        # Main portfolio
    ├── admin.html        # Admin dashboard
    ├── sw.js             # Service worker
    ├── manifest.json     # PWA config
    ├── css/              # Stylesheets
    └── js/               # JavaScript (main.js, admin.js)
```

## 🎨 Customization

### 🎨 Colors
Edit `src/css/main.css`:
